# 28 — Admin Frontend (Planned)

**Status:** ⬜ Planned · **Depends on:** Frontend Platform, Backend API · **Path:** `apps/dashboard`

## Scope
Planned admin experience for school operations. This app is the main management console for school administrators and finance staff.

## Planned Features
- school settings and module toggles
- academic year/class/section management
- student and staff management
- fee and payment workflows
- attendance and mark entry
- announcements and reports
- user and role administration

## Notes
The admin experience is intended to be the primary operational app and should be built on top of the backend modules already documented in this project.
