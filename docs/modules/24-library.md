# 24 — Library

**Status:** ⬜ Planned · **Depends on:** Student, Staff · **Path:** `app/Modules/Library`

## Scope
Planned optional module for library operations such as book cataloging, circulation, member management, and borrowing history.

## Planned Tables
| Table | Purpose / key columns |
|---|---|
| `books` | library inventory and metadata |
| `library_members` | student/staff membership records |
| `borrow_records` | lending and return lifecycle |

## Planned API Endpoints
- Admin endpoints for books and catalog management
- Member enrollment and circulation endpoints
- Borrow/return and due-date workflows

## Notes
This module is referenced in the optional module settings and build-order plan but does not yet have an implementation in the current workspace.
