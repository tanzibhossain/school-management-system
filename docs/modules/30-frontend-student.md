# 30 — Student Frontend (Planned)

**Status:** ⬜ Planned · **Depends on:** Frontend Platform, Backend API · **Path:** `apps/dashboard`

## Scope
Planned student-facing experience for viewing profile information, attendance, marks, invoices, announcements, and leave requests.

## Planned Features
- personal profile and academic history
- attendance dashboard
- marks and result view
- invoice and payment history
- announcement feed
- leave request submission

## Notes
This app should be a student-specific portal rather than a generic dashboard shell.
