# 25 — Transport

**Status:** ⬜ Planned · **Depends on:** Student, Payment · **Path:** `app/Modules/Transport`

## Scope
Planned optional module for student transport routing, vehicle assignment, route management, and fee-linked transport services.

## Planned Tables
| Table | Purpose / key columns |
|---|---|
| `transport_routes` | route definitions |
| `transport_vehicles` | vehicle inventory |
| `student_transport_assignments` | student-to-route/vehicle assignments |

## Planned API Endpoints
- Route and vehicle management endpoints
- Student assignment and route-change endpoints
- Fee and billing integration endpoints

## Notes
Transport-related reference data already exists in the Academic module, but a dedicated transport management module is still pending.
