# 29 — Teacher Frontend (Planned)

**Status:** ⬜ Planned · **Depends on:** Frontend Platform, Backend API · **Path:** `apps/dashboard`

## Scope
Planned teacher-facing experience for classroom operations, attendance, marks, routine, leave, and communications.

## Planned Features
- class routine and teaching assignment view
- student attendance entry
- mark entry and result review
- leave request review and approval workflows
- announcements and school notices

## Notes
This experience should be role-scoped and tailored to teacher workflows rather than exposing the full admin console.
