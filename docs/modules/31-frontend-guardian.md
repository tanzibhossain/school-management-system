# 31 — Guardian Frontend (Planned)

**Status:** ⬜ Planned · **Depends on:** Frontend Platform, Backend API · **Path:** `apps/dashboard`

## Scope
Planned guardian-facing experience for monitoring children’s attendance, marks, invoices, and announcements.

## Planned Features
- linked child overview
- child attendance and academic progress
- fee and invoice visibility
- school announcements
- leave-related communication and updates

## Notes
This app should support parent/guardian-specific access and keep the school data scoped to the children linked to the guardian account.
